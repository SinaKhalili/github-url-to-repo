Github URL to new repo

Simply type in what would be the GitHub URL if you were to make the repo, and you'll be redirected to the page to make it (instead of the 404).

Example: I want a repo named `beans`, so I go to the (non-existant)

https://github.com/sinakhalili/beans

And the 404 page will redirect to the new repo page!
